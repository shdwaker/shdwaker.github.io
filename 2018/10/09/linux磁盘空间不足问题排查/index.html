linux磁盘空间不足问题排查

发表于 2018-10-09 |

利用df、du等命令排查linux文件系统磁盘空间不足问题

背景

系统告警提示某台机器磁盘可用空间不足

排查过程

  • 登录机器

  • 列出文件系统的整体磁盘使用量

  • df显示 / 目录下磁盘使用率最高

  • cd /
  • 查看当前目录下每个文件夹的大小

如果当前目录下有隐藏文件，可以使用命令du -sh .[!.]*

.[!.] 是正则式, 意思是第一位是点, 第二位是除了点以外的字符, 第三位是任意字符或者不存在，后面可以再加一个” “来包括非隐藏文件

  • 进入空间最大的目录，继续du -sh *，直到找到占用空间最大的目录或文件

相关命令

df命令

列出文件系统的整体磁盘使用量

du命令

评估文件系统的磁盘使用量（常用于评估目录所占容量）

与df不一样的是，du会直接到文件系统内查找所有的文件数据。

# linux # df # du
